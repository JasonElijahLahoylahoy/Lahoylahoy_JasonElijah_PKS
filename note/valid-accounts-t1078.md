# Initial Access (TA0001) - Valid Accounts (T1078)

Initial Access (TA0001)

## Overview
Attackers abuse legitimate credentials to gain access, persist in networks, evade defenses, or escalate privileges.

## Why Valid Accounts Are Dangerous
- Bypass traditional perimeter defenses
- Appear as legitimate logins
- Enable persistent access
- Harder to detect

---

# Attack Vectors

## Sources of Credentials
- Data breaches
- Phishing attacks
- Third-party exposure
- Weak passwords

## Methods Used
- Credential stuffing
- Password spraying
- Session hijacking

---

# T1078 Account Types

## T1078.001 – Default Accounts
Built-in or factory-default accounts.

## T1078.002 – Domain Accounts
Accounts managed through Active Directory.

## T1078.003 – Local Accounts
Local machine accounts.

## T1078.004 – Cloud Accounts
Accounts associated with cloud services.

---

# Threat Groups Using T1078

## LAPSUS$
Uses stolen credentials and session tokens.

## Sandworm Team
Uses valid accounts for lateral movement.

## APT28
Uses credential theft and spearphishing.

## Volt Typhoon
Maintains persistence through legitimate credentials.
