# Certification Pathway

Passing certain cybersecurity certifications may grant a perfect score for quizzes and exams.

## Accepted Certifications

- ISC2 Certified in Cybersecurity
- CompTIA Security+
- Hack The Box Certifications
- TryHackMe Professional Certificates
- TCM Security Certifications

# Project: Personal Knowledge System (PKS)

A PKS is a structured digital notebook containing notes, research, and connected knowledge.

## Midterm Requirements

- Submit as `.zip`
- Markdown (`.md`) format only
- Fully self-contained
- Compatible with Obsidian vaults

## Required Files

- `START HERE.md` or `HOME.md`
